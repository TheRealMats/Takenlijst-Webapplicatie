# Takenlijst-Webapplicatie
Dit is een eenvoudige webapplicatie waarmee je taken en categorieën kunt bijhouden. Je kunt taken toevoegen, bekijken, als afgerond markeren en verwijderen. Hetzelfde geldt voor categorieën.

## Wat heb je nodig?
Zorg dat Python op je computer staat. Daarna installeer je de benodigde packages via de terminal:

-> pip install flask flask-sqlalchemy flask-wtf

## Hoe start je de applicatie?
1. Open een terminal in de projectmap
2. Typ het volgende commando: python app.py
3. Open je browser en ga naar http://127.0.0.1:5000

## Wat kun je doen in de applicatie?
- Een taak toevoegen met een titel, beschrijving en categorie
- Taken bekijken in het taakoverzicht
- Een taak als afgerond markeren of weer op bezig zetten
- Een taak verwijderen
- Een categorie toevoegen
- Categorieën bekijken en verwijderen

## Technologieen gebruikt
- Python & Flask
- SQLAlchemy (database)
- Flask-WTF (formulieren en beveiliging)
- Bootstrap 5 (opmaak)
- SQLite (database bestand)
